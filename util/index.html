    umask

    umask(mask)

    temporarily change umask

    Arguments

    • mask : a umask (invese of chmod argument)

    Example

    with umask(0o077), open('yay.txt') as f:
    f.write('nyaroo~n')

    yay.txt will be written with 600 file mode
